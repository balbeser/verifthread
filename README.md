Фреймворк - **Aiogram 3x**
База данных - **PostgreSQL**
Конфиг - **.env**


#### для запуска на Windows
1. Заполнить все поля в .env
2. Открыть консоль с проектом
3. Прописать команды
- `python -m venv venv`
- `.\venv\Scripts\activate`
- `pip install -r requirements.py`
- `python bot.py`
4. Бот запущен!
